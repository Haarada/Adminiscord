# Adminiscord
Adminiscord - Discord bot that makes administraiting servers easier

bot commands:
<br>
~~serverdata [enable/disable]
<br>
~~autokick [enable/disable]
<br>
~~ignoreuser [add/remove] [user_id]
<br><br>
